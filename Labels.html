    Labels - visual indexing...

    Label pages with one or more visual tags Select one or more visual tags and either...
    label the current web page with the tag(s) or...
    find page(s) labeled with the selected tag(s).
    Example: Tag a petshop webpage with and to find it among all your shopping sites, any pages about pets or, if you tap both the shopping and the pet icons, this specific site.

    This extension can be added to the browser on a computer and on a mobile device using a browser which allows extensions, such as Kiwi. Optionally, adding the Floccus extension in Chrome or Floccus app on a mobile device allows pages to be synced between devices - select the Stella bookmarks folder.
